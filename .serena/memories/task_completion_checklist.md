# タスク完了時のチェックリスト

## コード変更後の確認事項

### 1. 動作確認
- [ ] Chrome拡張機能をリロード（`chrome://extensions/` → 更新ボタン）
- [ ] YouTubeページで機能をテスト
- [ ] ポップアップUIが正常に動作するか確認
- [ ] シアターモードのオン/オフ切り替えを確認
- [ ] 透明度調整が正常に動作するか確認
- [ ] キーボードショートカット（Ctrl+Shift+T）が動作するか確認

### 2. テスト実行
```bash
# 全ユニットテスト実行
node test/run-all-unit-tests.js

# 関連するテストファイルを実行
node test/run-<component>-tests.js
```

### 3. コンソールエラー確認
- F12でDevToolsを開く
- Consoleタブでエラーがないか確認
- "YouTube Theater Mode:" でフィルタしてログ確認

### 4. 複数環境テスト
- 通常のYouTube動画ページ
- YouTube Shorts
- YouTube Live配信
- シアターモード（YouTubeの元々の機能）との併用

### 5. アクセシビリティ確認
- キーボードのみで操作可能か
- スクリーンリーダーで適切に読み上げられるか

### 6. 国際化確認（i18n変更時）
- `_locales/` 内の全言語ファイルを更新
- 新しいメッセージキーがすべての言語に存在するか確認

## コミット前の確認
- [ ] 不要なconsole.logを削除
- [ ] デバッグコードを削除
- [ ] コードスタイルが一貫しているか確認
- [ ] 変更内容が意図通りか最終確認
