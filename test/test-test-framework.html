TestFramework について

このテストフレームワークは、YouTube Theater Mode 拡張機能の開発とテストを支援するために設計されました。 Chrome API のモック機能、DOM テスト環境、包括的なアサーション機能を提供します。

Chrome API モック

chrome.storage、chrome.tabs、chrome.runtime API の完全なモック実装

DOM テスト環境

YouTube ページ構造のシミュレーションとテスト用要素の提供

アサーション機能

豊富なアサーション関数と詳細なエラーレポート

テストライフサイクル

beforeEach、afterEach、beforeAll、afterAll フック

エラーハンドリング

テスト失敗時の詳細な情報とスタックトレース

タイムアウト制御

テストごとのタイムアウト設定と長時間実行の防止
